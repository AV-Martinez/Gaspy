Gaspy

A gas station finder

Introduction

This Python application analyzes the prices of gas stations around a location or along a route. It will use public data about gas stations locations and prices to calculate the total cost of a refill based on the distance to the station, its current prices, the vehicle's consumption profile and any discounts available.

Things to know:

  • Prices source: This is the source for the data containing the characteristics of the gas stations, including at least its location and fuel prices.
  • Fuel type: Each price source refers to each fuel type in a specific way. While the fuels are basically the same across the globe, the name by which they are known differs by country.
  • Reference: Either a Location (specified as a latitude and a longitude) or a GPX file holding a Route. The search of gast stations will be performed around a location or along a route.
  • Vehicle profile: Specifies a consumption profile for a given vehicle:t Tank size (in litres), consumption (in litres per 100 Km) and fuel type (as available in the prices sources).
  • Discount: Specifies a price reduction percentage for a given fuel type. The discount will be applied to those gas stations which name contains the discount name.

Operation

All operation happens from the main screen. Before performing a search, you must specify:

  • The reference: Either a location from the dropdown list or a route, by choosing a GPX file.
  • The maximum distance to the reference: For a location reference, it will define a maximum circle to search for stations. For a route reference, it will define the maximum distance to the route track.
  • The vehicle's profile from the available options in the dropdown list.
  • Either to download fresh prices data or use the available ones. Note: Regardless of this choice, if the existing data is older than the number of hours specified in "Setup", fresh data will be downloaded.
  • The brands you want to show separated by a blank space. If you prefer to show all brands available, leave the field blank.

After clicking on "Search", the program will explore the available gas stations around the location or along teh route, filtering out those not fulfilling the conditions. The results table can be sorted on any column by clicking on the corresponding header.

The data shown provides:

  • Distance to the reference. In case of a route reference, the point in the route in which the station is located is also shown.
  • Total cost of refill, considering the vehicle fuel specified in the profile, its tank capacity and the consumption incurred to and from the reference, plus any discount available.
  • The brand of the station.
  • The date in which the prices used where valid.
  • The price used, which will match the fuel type in the vehicle's profile. Note that, if a discount is applied, the original price will appear next to the applied one.
  • The address.

Clicking right on any of the result lines, a popup menu will show these options:

  • Cost breakdown: Allows to understand the impact on the total refill cost of the need to cover the distance to the gas station.
  • Show on map: Will open a Google Maps map on the default browser centered at the gas station.
  • Directions: When the reference is around a location, this option will open a Google Maps route to reach the gas station.

Note on distances: The distance shown is an approximation based on the straight line distance between the gas station and the reference increased by a percentage as specified in "Setup". The final real distance will depend on the chosen route. You may want to fine tune the corrective percentage specified in "Setup" after checking some of the routes offered by the "Directions..." option.

The application saves the maximum distance to the reference used for each reference. You may edit the value offered at any time. Similarly, any brand name specified for filtering will be saved.

GPX files can be generated from Google Maps routes in http://www.mapstogpx.com. The "Info..." button will show the waypoints stored in the GPX file.

Setup

The "Setup" dialog allows to specify the Locations, Vehicle Profiles and Discounts that will be available. Click right on the corresponding tab content for a context menu from which you can create, edit or delete the corresponding items.

Additionally, you can specify:

  • The language to use. If changed, a restart will be needed.
  • The prices source to use. Note: The fuel type of the existing Vehicle Profiles may become invalid if it is not supported in the chosen source. A warning will alert of such circumstance. You must edit the impacted Profiles or revert to the previous source before leaving the dialog. You can also check the fuel type availability and the price ranges in the chosen source.
  • The maximum number of hours in which a downloaded prices file is considered valid. After this period of time, even requesting to use existing data from the main dialog will force a fresh data download.
  • The percentage to increase the straight line distance as to provide an approximate road route distance. See "Note on distances" above.
